pycpt
=====

Python tools to load and handle cpt (GMT format) color maps for use with matplotlib, e.g. from cpt-city

Installation
------------

```bash
pip install https://github.com/j08lue/pycpt/archive/master.zip
```

Tests
-----

You can run the tests with

```bash
pip install -e .[test]
pytest -v
```
